# 🔮 WHA: Alfabeto Mágico
**Herramienta Terapéutica para el Desarrollo de la Motricidad Fina**

Basada en el sistema de magia de *Witch Hat Atelier*, esta herramienta transforma la práctica de caligrafía en una experiencia encantadora. Diseñada para niños, enfatiza la "Bondad Absoluta" al transmutar los errores de trazo en recompensas visuales (estrellas, pétalos y partículas).

## ✨ Características
- **Interfaz "Zen" Minimalista:** Baja carga cognitiva, alta maravilla visual.
- **Dificultad Ajustable:** Slider de calibración dinámica (Fácil, Normal, Difícil) para adaptarse a la edad y nivel de habilidad motora del niño.
- **Detección Secuencial de Caminos:** Utiliza un sistema de "Zonas de Poder" vectoriales para asegurar que las letras se tracen en el orden correcto.
- **Refuerzo Fonético Afectuoso (TTS):** Una voz cálida y dulce felicita al niño y menciona la letra al completarla. Soporta múltiples idiomas (Español MX, Inglés).
- **Optimización Móvil:** Soporte táctil completo para smartphones y tablets (iPad/Android).
- **Optimizado para Wacom/Proyector:** Funciona perfectamente en "Artist Mode" para proyecciones inmersivas.

## 🛠️ Stack Tecnológico
- **HTML5 Canvas:** Física de partículas y dibujo en tiempo real.
- **Web Speech API:** Síntesis de voz multiplataforma.
- **CSS Responsivo:** Optimizado para dispositivos modernos.

## 🚀 Cómo Ejecutar (Autohospedado)
Esta es una aplicación web de un solo archivo. No requiere dependencias.
1. Descarga `index.html`.
2. Ábrelo en cualquier navegador moderno.
3. Haz clic en el sigilo de inicio para activar la magia y el sonido.
4. **Controles:**
   - **Dibujo:** Usa el mouse, el dedo o una pluma digital.
   - **Reset Mágico:** Usa el botón de marfil en la parte inferior para volver a empezar.
   - **Slider de Dificultad:** Calibra la sensibilidad de detección en tiempo real.
